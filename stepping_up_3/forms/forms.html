Forms and buttons!

Linking the lable to the input element

The lable element

In the html label element we link the label to the input field by selecting a namefor the label wiht for="text".

The input element

The same lable name "text" is now set to the inputs id name: id="text" so that the label and input are linked toghether!

The different forms

When value="Enter your first name" is chosen its plain text that can be changed by the user.

When placeholder="Enter your last name" is chosen its a hint to the user of what kind of input goes here!
There is also more opacity on the placeholder text!

The "aria-label" lets the assistive technology know the label of the form without a lable showing!

Buttons

When the user clicks the button the form is performing the action that we have chosen for the form: action="home.html"

There is a "reset" button that deletes everything in the form. It considered bad practice to use this though! The users usually just want to change something in the form and not delete everything!
